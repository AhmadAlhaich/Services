***HOW TO LAUNCH AND USE WEBSITE***

1. Extract the files from the Project.zip
2. Launch WAMP Server.
3. Add files, "cleaning" "connect2" "internet" 
   "plumbing 2.0" "repair", into the "www" folder
   in the wamp server directory.
4. Go to your browser and type "Localhost/connect2"
   to launch the website.
5. to test database connectivity:
   -use sign up/in pages
   -use "privacy" option in main page (change 
    password, delete account).
   -use "plumbing" page (DrainAid, Wrench
   Companies).
